MEMORY GAME 

Memory game is card game in which all of the cards are laid face down on a surface 
and two cards are flipped face up over each turn. The object of the game is to 
turn over pairs of matching cards.

It's a game built for practice simple web application, using HTML, CSS, JavaScript and Python(Flask).
